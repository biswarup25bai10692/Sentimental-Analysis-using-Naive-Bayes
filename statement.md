In the world we live in people are always sharing what they think online through reviews, comments and posts on media. These thoughts can tell us a lot about how people feel about something. Whether they like it do not like it. Feel okay about it.. It is hard to read thousands of these texts by hand to see what people think overall. It takes a lot of time. It is not easy to do.

The big problem is to make a system that can quickly and easily understand and categorize what people think about something. The Sentiment Analysis system should be able to take what a user says look at it and figure out if the user likes something does not like it or feels okay about it without a person having to do it each time. The Sentiment Analysis system should also be simple to use and give results.

To fix this problem this project makes a Sentiment Analysis model using the Naive Bayes algorithm. The Sentiment Analysis system first makes the text simpler by making it all lowercase and removing punctuation that is not needed. Then it turns the text into numbers using a method called Bag of Words so the machine learning model can understand the text.

When the Sentiment Analysis model is trained on some sample data it can look at sentences that a user enters and guess how the user feels about something. The Sentiment Analysis model also gives a score that shows how sure it is about what it thinks.

Even though the Sentiment Analysis model uses an amount of data it clearly shows how machine learning can help us understand what people are saying. This project is a good step, towards making better systems that can look at a lot of text by themselves.
